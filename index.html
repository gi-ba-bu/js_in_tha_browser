Git commands for beginners

Three popular commands

git init

This is used to set up the current folder as a github repository

git checkout -b <branch name>

This is used to create a new branch and make this new branch the currently active branch.

git clone <remote repository>

This is used to clone a remote repository. If multiple brnaches are present, a remote connection is established with all branches. Only one branch will be checked out. Other branches will need to be pulled.
